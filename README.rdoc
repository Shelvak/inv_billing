= INV Billing

- Script para generar planillas MV-0x del Instituto Nacional de Vitivinicultura.
- Se diferenciaran por carpetas y nombres de inscriptos

```
  $HOME/planillas/abril-2013/Bodega-buena-onda.csv
```
